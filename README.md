
## Usage

Add `110515` to the plugins section of your `.eslintrc` configuration file. You can omit the `eslint-plugin-` prefix:

```json
{
    "plugins": [
        "110515"
    ]
}
```


Then configure the rules you want to use under the rules section.

```json
{
    "rules": {
        "110515/rule-name": 2
    }
}
```
